
# Original 查看原文
### 介绍

Original 是各种题型包含的音频查看原文组件

### 引入

```js
import Vue from 'vue';
import { Original } from 'queens';

Vue.use(Original);
```

## 代码演示

### 基础用法

```html
<original content="What is your name?"></original>
```

## API

### Props

| 参数 | 说明 | 类型 | 默认值 |
|------|------|------|------|
| content | 原文内容 | *string* | `-` |
